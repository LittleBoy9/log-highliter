Features

Everything You Need

Built to make debugging faster and codebase cleanup effortless.

Color-Coded Highlights

Each console method gets its own distinct color. Instantly see the difference between logs, warnings, errors, info, and debug statements.

One-Click Removal

Remove all highlighted log statements from your code with a single click. Built-in confirmation dialog keeps you safe from accidents.

Framework Presets

Built-in support for NestJS, Winston, Pino, Bunyan, and Log4js. Enable presets with a single toggle and highlight framework-specific patterns.

Smart Parsing

Handles multi-line statements, nested parentheses, template literals, and comments. Never miss a complex console call.

Custom Patterns

Define your own logging patterns with custom colors via settings.json. Match any function call — myLogger.log, debugPrint, and more.

Multi-Language

Works with JavaScript, TypeScript, React (JSX/TSX), Vue, and Svelte files. Automatically activates when you open a supported file.

Configuration

Fully Customizable

Configure presets and add custom patterns directly in your settings.json.

Enable Framework Presets
{
  "logHighlighter.enabledPresets": [
    "nestjs",
    "winston",
    "pino"
  ]
}
Add Custom Patterns
{
  "logHighlighter.customPatterns": [
    {
      "name": "My Logger",
      "pattern": "myLogger.log",
      "color": "#ff6b6b"
    }
  ]
}
